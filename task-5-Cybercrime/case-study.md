# Cybercrime Case Study

## Introduction
Cybercrime refers to illegal activities carried out using the internet or digital devices. It has become very common in today’s digital world.

## Case Study: Phishing Attack
A common cybercrime example is phishing. In phishing, attackers send fake emails or messages pretending to be from trusted sources like banks or companies.

For example, a user receives a message saying their bank account will be blocked. The message contains a link that looks like a real bank website. When the user enters their login details, the attacker steals their information.

## Impact
- Financial loss
- Account hacking
- Identity theft
- Loss of personal data

## Conclusion
Cybercrime can affect anyone. Awareness and careful online behavior can help prevent such attacks.
